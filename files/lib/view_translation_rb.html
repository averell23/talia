extract_translations_to_update(translations)

Extract and normalize translations to pass them to ActiveRecord#update. It select translations with an id in order to extract and made it an hash key, useful for ActiveRecord#update.

ActiveRecord#update accepts two params, an Array of ids and another one, composed by the values to be updated, formatted in an Hash.

Example: 

  [ { "id" => "1", "tr_key" => "hello", "text" => "Hello!" },
    { "id" => "2", "tr_key" => "rabbit", "text" => "Rabbit" } ]

  # => [ [1,2], [ {"tr_key" => "hello", "text" => "Hello!" },
                 { "tr_key" => "rabbit", "text" => "Rabbit" } ] ]

find_all_for(key)

Find all translations for the given key. This will return a hash that contains all language codes enabled in the system as keys, and the value will be a hash where :translation points to the translation object and :lang_name to the language name
