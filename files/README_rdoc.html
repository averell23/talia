Talia

This application is used for the current development version of Talia. If you want to get started, you may want to check out the documentation for the "core" classes instead:

net7.github.com/talia_core/

More documentation can be found on the Talia website: net7sviluppo.com/trac/talia/

You may also want to consult the API documentation at net7.github.com/talia/

Installation and Configuration

See the guide on the Wiki on how to create a new application with the Talia core classes (which is probably what you want to do)

net7sviluppo.com/trac/talia/wiki/TaliaInstallation

You may also clone this sample application to play around with; and the user from the Discovery project can check their specialized installation guide:

net7sviluppo.com/trac/talia/wiki/InstallTaliaForDiscoveryPartners
