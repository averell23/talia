import(base_uri, list_location = "?getList=all", sig_request = "?get=", file_ext='')

Import a single source from XML. The xml_source is the source of the element to import. If this is an URL, it will be used to fetch the information from the web.

The base_uri is the uri where the import data is retrieved. The list_location and sig_request are additional uri parts that are appended to the URI to retrieve the list of all sigla and the documents themselves.

To retrieve a document, the siglum from the list xml will be appended to base_uri + sig_request.
